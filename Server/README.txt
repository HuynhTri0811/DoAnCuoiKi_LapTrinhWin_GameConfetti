1.Name
	Server game confitti

2.Information
	The server sends questions and processes the responses that the client submits

3.WHAT DO YOU NEED ?
	*You are using Windows 7 or higher . Linux and MacOs not currently supported

4.Setup
	Unzip the DoAnCuoiKi_LapTrinhWin_GameConfetti.zip file and in the bin/Debug directory, find server.exe and start running

5.How to use
	1 . you start server
	2 . Wait for more clients to come in game
	3 . Choose question in "Danh sách câu hỏi"
	4 . Send question to client
	5 . Send answer to client
	6 . When Question's ten , server send client who win
